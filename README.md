## About
* In this challenge, I used two separate APIs together to create an IP address tracking application: to get the IP Address locations, i used the <a href="https://geo.ipify.org/">IP Geolocation API by IPify<a/>; to generate the map, <a href="https://leafletjs.com/">LeafletJS<a/>.
* I also used <a href="https://axios-http.com/docs/intro">Axios</a>, a Promise based HTTP client for the browser and node.js

## Features
The main features of the application are:
* See your own IP address on the map on initial page load.
* Search for any IP address or domain to see key information and location.
